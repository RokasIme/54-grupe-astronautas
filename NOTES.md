# Terminalo komandos

## touch

sukuriamas vienas failas:

```
touch index.html
```

Sukuriami keli failai:

```
touch main.css button.css layout.css
```

## ls

stulpeliu išspausdina esamojoje direktorijoje esantį turinį: direktorijas (folderius) ir failus

```
ls -l
```
